## Overview

MUI's Skeleton component provides placeholder animations to display while content is loading. It can represent loading states for various elements such as text, images, and cards, visually communicating to users that content is loading. By combining multiple variations (text, circular, rectangular, rounded) with animation effects, you can achieve a sophisticated loading experience.

</En>
<Ja>

## 概要

MUIのSkeletonコンポーネントは、コンテンツの読み込み中に表示するプレースホルダーアニメーションを提供します。テキスト、画像、カードなど、さまざまな要素のローディング状態を表現でき、コンテンツが読み込まれていることを視覚的にユーザーに伝えます。複数のバリエーション（テキスト、円形、矩形、角丸）をアニメーション効果と組み合わせることで、洗練されたローディング体験を実現できます。

</Ja>

<Ja>

## 使用例

</Ja>

```tsx
import Skeleton from '@mui/material/Skeleton'
import Stack from '@mui/material/Stack'

function LoadingState() {
  return (
    <Stack spacing={1}>
      <Skeleton variant="text" sx={{ fontSize: '1rem' }} />
      <Skeleton variant="circular" width={40} height={40} />
      <Skeleton variant="rectangular" width={210} height={60} />
      <Skeleton variant="rounded" width={210} height={60} />
    </Stack>
  )
}
```

## Accessibility

- Communicates loading state to screen readers
- Animation respects `prefers-reduced-motion`
- Recommend using `aria-busy` attribute to explicitly indicate loading state

</En>
<Ja>

## アクセシビリティ

- スクリーンリーダーにローディング状態を伝達
- アニメーションは`prefers-reduced-motion`を尊重
- ローディング状態を明示的に示すために`aria-busy`属性の使用を推奨

</Ja>
