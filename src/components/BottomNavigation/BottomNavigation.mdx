# BottomNavigation

Navigation at the bottom of the screen

## Overview

The BottomNavigation component provides a navigation bar positioned at the bottom of mobile applications. It allows quick switching between 3-5 primary destinations. Ideal for mobile-first design, positioning navigation in an easily accessible location for finger interaction.

Key features:
- 3-5 navigation actions
- Display icons and labels
- Visual feedback for active state
- Fixed position display support
- Responsive design

# BottomNavigation

画面下部のナビゲーション

## 概要

BottomNavigationコンポーネントは、モバイルアプリケーションの下部に配置されるナビゲーションバーを提供します。3〜5つの主要な目的地間での素早い切り替えを可能にします。モバイルファーストデザインに最適で、指での操作がしやすい位置にナビゲーションを配置します。

主な機能:
- 3〜5つのナビゲーションアクション
- アイコンとラベルの表示
- アクティブ状態の視覚的フィードバック
- 固定位置表示のサポート
- レスポンシブデザイン

## Usage Example

```tsx
import BottomNavigation from '@mui/material/BottomNavigation'
import BottomNavigationAction from '@mui/material/BottomNavigationAction'
import RestoreIcon from '@mui/icons-material/Restore'
import FavoriteIcon from '@mui/icons-material/Favorite'
import LocationOnIcon from '@mui/icons-material/LocationOn'
import React from 'react'

function MyBottomNavigation() {
  const [value, setValue] = React.useState(0)

  return (
    <BottomNavigation
      showLabels
      value={value}
      onChange={(event, newValue) => {
        setValue(newValue)
      }}
    >
      <BottomNavigationAction label="Recents" icon={<RestoreIcon />} />
      <BottomNavigationAction label="Favorites" icon={<FavoriteIcon />} />
      <BottomNavigationAction label="Nearby" icon={<LocationOnIcon />} />
    </BottomNavigation>
  )
}

// Fixed position BottomNavigation
function FixedBottomNavigation() {
  const [value, setValue] = React.useState(0)

  return (
    <Paper sx={{ position: 'fixed', bottom: 0, left: 0, right: 0 }} elevation={3}>
      <BottomNavigation
        showLabels
        value={value}
        onChange={(event, newValue) => setValue(newValue)}
      >
        <BottomNavigationAction label="Home" icon={<RestoreIcon />} />
        <BottomNavigationAction label="Favorites" icon={<FavoriteIcon />} />
        <BottomNavigationAction label="Nearby" icon={<LocationOnIcon />} />
      </BottomNavigation>
    </Paper>
  )
}
```

## Accessibility

- Each BottomNavigationAction has a `button` role
- The currently selected action has `aria-current="page"` set
- Provide meaningful `label` for each action
- Keyboard navigation is fully supported
- Screen readers properly announce action states

</En>

## アクセシビリティ

- 各BottomNavigationActionには`button`ロールが設定されます

- キーボードナビゲーションは完全にサポートされています
- スクリーンリーダーはアクションの状態を適切にアナウンスします
