# FloatingActionButton

A Floating Action Button (FAB) is a circular button component that represents the most important action on the screen.

## Overview

FloatingActionButton is a special button used to promote the primary action of an application. Typically fixed to the bottom right of the screen, it provides immediate access to the action users perform most frequently.

Key features:
- **Prominent Design**: Circular and elevated design visually distinguishes it from other content
- **Icon Display**: Places an icon representing the action in the center
- **Extended Variant**: Can combine icon and text label in extended display
- **Color Variations**: Supports theme colors like primary, secondary
- **Disabled State**: Can restrict operation with the disabled property

Common use cases:
- Create new (email, post, document, etc.)
- Transition to edit mode
- Add to favorites
- Navigation actions

# FloatingActionButton

Floating Action Button (FAB) は、画面上で最も重要なアクションを表す円形のボタンコンポーネントです。

## 概要

FloatingActionButton は、アプリケーションのプライマリアクションを促進するために使用される特別なボタンです。通常、画面の右下に固定され、ユーザーが最も頻繁に実行するアクションへの即座のアクセスを提供します。

主な機能:
- **目立つデザイン**: 円形で浮き上がったデザインが他のコンテンツと視覚的に区別されます
- **アイコン表示**: アクションを表すアイコンを中央に配置します
- **拡張バリアント**: アイコンとテキストラベルを組み合わせた拡張表示が可能です
- **カラーバリエーション**: primary、secondary などのテーマカラーをサポート
- **無効状態**: disabled プロパティで操作を制限できます

一般的な使用例:
- 新規作成 (メール、投稿、ドキュメントなど)
- 編集モードへの遷移
- お気に入りへの追加
- ナビゲーションアクション

## Usage Example

```tsx
import AddIcon from '@mui/icons-material/Add'
import Fab from '@mui/material/Fab'

// Basic usage
<Fab color="primary" aria-label="add">
  <AddIcon />
</Fab>

// Extended variant (icon and text)
<Fab variant="extended" color="primary">
  <NavigationIcon sx={{ mr: 1 }} />
  Navigate
</Fab>

// Disabled state
<Fab disabled aria-label="like">
  <FavoriteIcon />
</Fab>
```

## Accessibility

- Always specify the `aria-label` attribute to clarify the button's purpose
- Particularly important for icon-only buttons (for screen reader users)
- Supports keyboard operation: Tab key to focus, Enter or Space key to execute
- When disabled, `aria-disabled="true"` is automatically set
</En>

<Ja>
## 使用例

```tsx
import AddIcon from '@mui/icons-material/Add'
import Fab from '@mui/material/Fab'

// 基本的な使用
<Fab color="primary" aria-label="追加">
  <AddIcon />
</Fab>

// 拡張バリアント (アイコンとテキスト)
<Fab variant="extended" color="primary">
  <NavigationIcon sx={{ mr: 1 }} />
  ナビゲート
</Fab>

// 無効状態
<Fab disabled aria-label="お気に入り">
  <FavoriteIcon />
</Fab>
```

## アクセシビリティ

- 必ず `aria-label` 属性を指定してボタンの目的を明確にしてください
- アイコンのみのボタンでは特に重要です (スクリーンリーダーユーザーのため)
- キーボード操作をサポート: Tab キーでフォーカス、Enter または Space キーで実行
- 無効化された場合、`aria-disabled="true"` が自動的に設定されます
</Ja>
