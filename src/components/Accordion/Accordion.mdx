# Accordion

Collapsible content panels

## Overview

The Accordion component provides vertically stacked content panels. Each panel can be expanded and collapsed, making it ideal for organizing and displaying large amounts of information. Commonly used for FAQs, settings screens, and detailed information displays.

Key features:
- Independent expand/collapse of multiple panels
- Controlled accordion (only one panel expanded at a time)
- Customizable icons and styles
- Support for disabled state
- Configure default expanded panels

# Accordion

折りたたみ可能なコンテンツパネル

## 概要

Accordionコンポーネントは、垂直に積み重ねられたコンテンツパネルを提供します。各パネルは展開・折りたたみが可能で、大量の情報を整理して表示するのに最適です。FAQ、設定画面、詳細情報の表示によく使用されます。

主な機能：
- 複数パネルの独立した展開・折りたたみ
- 制御されたアコーディオン（一度に1つのパネルのみ展開）
- カスタマイズ可能なアイコンとスタイル
- 無効状態のサポート
- デフォルトで展開するパネルの設定

## Usage Example

```tsx
import Accordion from '@mui/material/Accordion'
import AccordionDetails from '@mui/material/AccordionDetails'
import AccordionSummary from '@mui/material/AccordionSummary'
import ExpandMoreIcon from '@mui/icons-material/ExpandMore'
import Typography from '@mui/material/Typography'

function MyAccordion() {
  return (
    <div>
      <Accordion>
        <AccordionSummary expandIcon={<ExpandMoreIcon />}>
          <Typography>Section 1</Typography>
        </AccordionSummary>
        <AccordionDetails>
          <Typography>
            Content goes here.
          </Typography>
        </AccordionDetails>
      </Accordion>

      <Accordion>
        <AccordionSummary expandIcon={<ExpandMoreIcon />}>
          <Typography>Section 2</Typography>
        </AccordionSummary>
        <AccordionDetails>
          <Typography>
            Multiple accordions can be placed.
          </Typography>
        </AccordionDetails>
      </Accordion>
    </div>
  )
}
```

## Accessibility

- AccordionSummary automatically has a `button` role and supports keyboard operation
- The `aria-expanded` attribute communicates the panel's expansion state to screen readers
- The `aria-controls` attribute can be used to explicitly associate the summary with its content
- Panels can be expanded/collapsed using the Enter or Space key
</En>

## 使用例

```tsx
import Accordion from '@mui/material/Accordion'
import AccordionDetails from '@mui/material/AccordionDetails'
import AccordionSummary from '@mui/material/AccordionSummary'
import ExpandMoreIcon from '@mui/icons-material/ExpandMore'
import Typography from '@mui/material/Typography'

function MyAccordion() {
  return (
    <div>
      <Accordion>
        <AccordionSummary expandIcon={<ExpandMoreIcon />}>
          <Typography>セクション 1</Typography>
        </AccordionSummary>
        <AccordionDetails>
          <Typography>
            コンテンツがここに入ります。
          </Typography>
        </AccordionDetails>
      </Accordion>

      <Accordion>
        <AccordionSummary expandIcon={<ExpandMoreIcon />}>
          <Typography>セクション 2</Typography>
        </AccordionSummary>
        <AccordionDetails>
          <Typography>
            複数のアコーディオンを配置できます。
          </Typography>
        </AccordionDetails>
      </Accordion>
    </div>
  )
}
```

## アクセシビリティ

- AccordionSummaryは自動的に`button`ロールを持ち、キーボード操作をサポートします
- `aria-expanded`属性がパネルの展開状態をスクリーンリーダーに伝えます
- `aria-controls`属性を使用してサマリーとコンテンツを明示的に関連付けることができます
- EnterキーまたはSpaceキーを使用してパネルを展開・折りたたむことができます
</Ja>
