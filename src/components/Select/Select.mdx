# Select

Select box input

## Overview

MUI's Select component is a form input component for selecting options from a dropdown menu. It supports single or multiple selection and provides various style and customization options. When used in combination with the MenuItem component, you can implement accessible selection forms.

## Usage Example

```tsx
import Select from '@mui/material/Select'
import MenuItem from '@mui/material/MenuItem'

function MySelect() {
  const [age, setAge] = useState(10)

  return (
    <Select
      id="age-select"
      value={age}
      label="Age"
      onChange={(e) => setAge(e.target.value)}
    >
      <MenuItem value={10}>Ten</MenuItem>
      <MenuItem value={20}>Twenty</MenuItem>
      <MenuItem value={30}>Thirty</MenuItem>
    </Select>
  )
}
```

## Accessibility

- Specify the `id` property to associate with label
- Provide visual label with `label` property
- Supports keyboard operations (arrow keys, Enter, Escape, etc.)
- Properly recognized by screen readers
</En>

## 使用例

```tsx
import Select from '@mui/material/Select'
import MenuItem from '@mui/material/MenuItem'

function MySelect() {
  const [age, setAge] = useState(10)

  return (
    <Select
      id="age-select"
      value={age}
      label="年齢"
      onChange={(e) => setAge(e.target.value)}
    >
      <MenuItem value={10}>10歳</MenuItem>
      <MenuItem value={20}>20歳</MenuItem>
      <MenuItem value={30}>30歳</MenuItem>
    </Select>
  )
}
```

## アクセシビリティ

- `id`プロパティを指定してラベルと関連付けます
- `label`プロパティで視覚的なラベルを提供します
- キーボード操作をサポートしています（矢印キー、Enter、Escapeなど）
- スクリーンリーダーによって適切に認識されます
