# Divider

A component that displays dividing lines to separate content. Visually divides sections and clarifies hierarchical structure.

## Overview

The Divider component wraps Material-UI's Divider and provides the following features:

- **Orientation**: `horizontal`, `vertical`
- **Variants**: `fullWidth`, `inset` (left indent), `middle` (centered)
- **With Text**: Display text or icons by passing children
- **Position**: Choose `left`, `center`, `right` with textAlign property
- **Light Version**: Display thin line with light property
- **flexItem**: Configuration for proper behavior in Flex containers

## Usage Example

```tsx
import Divider from '@/components/Divider'
import Box from '@mui/material/Box'
import Typography from '@mui/material/Typography'

// Basic horizontal divider
<Box>
  <Typography>Section 1</Typography>
  <Divider />
  <Typography>Section 2</Typography>
</Box>

// Divider with text
<Divider>OR</Divider>

// Divider with text alignment
<Divider textAlign="left">Left</Divider>
<Divider textAlign="center">Center</Divider>
<Divider textAlign="right">Right</Divider>

// Vertical divider
<Box sx={{ display: 'flex', alignItems: 'center', height: 100 }}>
  <Typography>Left</Typography>
  <Divider orientation="vertical" flexItem />
  <Typography>Right</Typography>
</Box>

// Inset divider (for lists)
<Divider variant="inset" />

// Middle divider
<Divider variant="middle" />

// Light divider
<Divider light />
```

## Accessibility

- Divider is recognized as `role="separator"`
- As a decorative divider, it's typically not announced by screen readers
- Consider using the `<hr>` element for semantic separation
- When using Divider with text, ensure the text is meaningful
</En>

## 使用例

```tsx
import Divider from '@/components/Divider'
import Box from '@mui/material/Box'
import Typography from '@mui/material/Typography'

// 基本的な水平区切り線
<Box>
  <Typography>Section 1</Typography>
  <Divider />
  <Typography>Section 2</Typography>
</Box>

// テキスト付き区切り線
<Divider>OR</Divider>

// テキスト配置付き区切り線
<Divider textAlign="left">Left</Divider>
<Divider textAlign="center">Center</Divider>
<Divider textAlign="right">Right</Divider>

// 垂直区切り線
<Box sx={{ display: 'flex', alignItems: 'center', height: 100 }}>
  <Typography>Left</Typography>
  <Divider orientation="vertical" flexItem />
  <Typography>Right</Typography>
</Box>

// インセット区切り線（リスト用）
<Divider variant="inset" />

// ミドル区切り線
<Divider variant="middle" />

// ライト区切り線
<Divider light />
```

## アクセシビリティ

- Dividerは`role="separator"`として認識されます
- 装飾的な区切り線として、通常スクリーンリーダーでは読み上げられません
- セマンティックな区切りには`<hr>`要素の使用を検討してください
- テキスト付きDividerを使用する場合、テキストが意味のあるものであることを確認してください
