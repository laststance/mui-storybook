# Breadcrumbs

Breadcrumb navigation

## Overview

The Breadcrumbs component is a navigation element that hierarchically displays the user's current location. It shows the current location within a website or application and provides navigation to higher levels. Also beneficial for SEO, helping convey site structure to crawlers.

Key features:
- Customizable separators
- Breadcrumbs with icons
- Set maximum display items (collapse functionality)
- Support for link and non-link items
- Automatic identification of active item

</En>

# Breadcrumbs

パンくずナビゲーション

## 概要

Breadcrumbsコンポーネントは、ユーザーの現在位置を階層的に表示するナビゲーション要素です。ウェブサイトやアプリケーション内での現在位置を示し、上位階層へのナビゲーションを提供します。また、SEOにも有益で、サイト構造をクローラーに伝えるのに役立ちます。

主な機能:
- カスタマイズ可能なセパレータ
- アイコン付きパンくず
- 最大表示項目数の設定（折りたたみ機能）
- リンクと非リンク項目のサポート
- アクティブ項目の自動識別

## Usage Example

```tsx
import Breadcrumbs from '@mui/material/Breadcrumbs'
import Link from '@mui/material/Link'
import Typography from '@mui/material/Typography'
import HomeIcon from '@mui/icons-material/Home'
import NavigateNextIcon from '@mui/icons-material/NavigateNext'

function MyBreadcrumbs() {
  return (
    <Breadcrumbs aria-label="breadcrumb">
      <Link underline="hover" color="inherit" href="/">
        Home
      </Link>
      <Link underline="hover" color="inherit" href="/products">
        Products
      </Link>
      <Typography color="text.primary">Details</Typography>
    </Breadcrumbs>
  )
}

// Breadcrumbs with icons
function IconBreadcrumbs() {
  return (
    <Breadcrumbs aria-label="breadcrumb">
      <Link
        underline="hover"
        sx={{ display: 'flex', alignItems: 'center' }}
        color="inherit"
        href="/"
      >
        <HomeIcon sx={{ mr: 0.5 }} fontSize="inherit" />
        Home
      </Link>
      <Typography
        sx={{ display: 'flex', alignItems: 'center' }}
        color="text.primary"
      >
        Products
      </Typography>
    </Breadcrumbs>
  )
}

// Custom separator
function CustomSeparator() {
  return (
    <Breadcrumbs
      separator={<NavigateNextIcon fontSize="small" />}
      aria-label="breadcrumb"
    >
      <Link underline="hover" color="inherit" href="/">
        Home
      </Link>
      <Link underline="hover" color="inherit" href="/category">
        Category
      </Link>
      <Typography color="text.primary">Product</Typography>
    </Breadcrumbs>
  )
}

// Collapse functionality
function CollapsedBreadcrumbs() {
  return (
    <Breadcrumbs maxItems={3} aria-label="breadcrumb">
      <Link underline="hover" color="inherit" href="/">
        Home
      </Link>
      <Link underline="hover" color="inherit" href="/category">
        Category
      </Link>
      <Link underline="hover" color="inherit" href="/subcategory">
        Subcategory
      </Link>
      <Link underline="hover" color="inherit" href="/product">
        Product
      </Link>
      <Typography color="text.primary">Details</Typography>
    </Breadcrumbs>
  )
}
```

## Accessibility

- Use `aria-label` to describe the purpose of the breadcrumbs
- The current page (last item) is typically displayed as Typography, not a Link
- Placing within a `<nav>` element makes it recognized as a navigation landmark
- Screen readers automatically announce separators
- Combining with structured data (JSON-LD) improves SEO

</En>
