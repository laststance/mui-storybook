# Modal

A modal overlay component that displays temporary content to capture user attention.

## Overview

The Modal component wraps Material-UI's Modal and provides the following features:

- **Overlay**: Blocks background to focus attention
- **Transitions**: Animation effects like fade and slide
- **Focus management**: Focus trap within modal
- **Escape key support**: Supports closing with Escape key

## Usage Example

```tsx
import Modal from '@mui/material/Modal'
import Box from '@mui/material/Box'
import Button from '@mui/material/Button'

const style = {
  position: 'absolute',
  top: '50%',
  left: '50%',
  transform: 'translate(-50%, -50%)',
  width: 400,
  bgcolor: 'background.paper',
  boxShadow: 24,
  p: 4,
}

function BasicModal() {
  const [open, setOpen] = React.useState(false)

  return (
    <>
      <Button onClick={() => setOpen(true)}>Open Modal</Button>
      <Modal
        open={open}
        onClose={() => setOpen(false)}
        aria-labelledby="modal-title"
      >
        <Box sx={style}>
          <h2 id="modal-title">Modal Title</h2>
          <p>Modal content goes here</p>
        </Box>
      </Modal>
    </>
  )
}
```

## Accessibility

- Use `aria-labelledby` and `aria-describedby` to describe content
- Focus automatically moves to and is trapped within the modal
- Modal can be closed with Escape key
- After modal closes, focus returns to the original element
