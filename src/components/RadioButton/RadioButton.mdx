# RadioButton

A radio button input component that provides form elements for selecting one option from multiple choices.

## Overview

The RadioButton component wraps Material-UI's Radio and RadioGroup and provides the following features:

- **Grouping**: Management of multiple radio buttons with RadioGroup
- **Colors**: `primary`, `secondary`, `default`
- **Sizes**: `small`, `medium`
- **With labels**: Integration with FormControlLabel

## 使用例

<En>
```tsx
import Radio from '@mui/material/Radio'
import RadioGroup from '@mui/material/RadioGroup'
import FormControlLabel from '@mui/material/FormControlLabel'
import FormControl from '@mui/material/FormControl'
import FormLabel from '@mui/material/FormLabel'

function BasicRadioButtons() {
  const [value, setValue] = React.useState('option1')

  return (
    <FormControl>
      <FormLabel>Select an option</FormLabel>
      <RadioGroup
        value={value}
        onChange={(e) => setValue(e.target.value)}
      >
        <FormControlLabel
          value="option1"
          control={<Radio />}
          label="Option 1"
        />
        <FormControlLabel
          value="option2"
          control={<Radio />}
          label="Option 2"
        />
        <FormControlLabel
          value="option3"
          control={<Radio />}
          label="Option 3"
        />
      </RadioGroup>
    </FormControl>
  )
}

// Custom colors and sizes
<Radio color="secondary" size="small" />
```

## Accessibility

- Use appropriate `aria-label` or `FormLabel` for RadioGroup
- Provide descriptive labels for each radio button
- Supports keyboard navigation (arrow keys, Tab)
- Supports form validation with `required` attribute
- Recommend having at least one option selected by default in the group
</En>

## アクセシビリティ

- RadioGroupには適切な`aria-label`または`FormLabel`を使用してください
- 各ラジオボタンに説明的なラベルを提供してください
- キーボードナビゲーション（矢印キー、Tab）をサポート
- `required`属性によるフォームバリデーションをサポート
- グループ内で少なくとも1つのオプションをデフォルトで選択することを推奨
