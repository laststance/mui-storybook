# Badge

Badges displaying notification counts or status

## Overview

The Badge component displays a small badge in the corner of icons or buttons to indicate unread message counts, status, etc. Used for notification systems, shopping cart item counts, status indicators, and more.

Key features:
- Numeric badges and dot badges
- Color variations (primary, secondary, error, success, etc.)
- Maximum value setting (99+, etc.)
- Badge position adjustment
- Control badge visibility
- Custom content support

## Usage Example

```tsx
import Badge from '@mui/material/Badge'
import IconButton from '@mui/material/IconButton'
import MailIcon from '@mui/icons-material/Mail'
import NotificationsIcon from '@mui/icons-material/Notifications'
import Stack from '@mui/material/Stack'

function MyBadges() {
  return (
    <Stack direction="row" spacing={3}>
      {/* Numeric badge */}
      <Badge badgeContent={4} color="primary">
        <MailIcon />
      </Badge>

      {/* Set maximum value */}
      <Badge badgeContent={100} max={99} color="secondary">
        <MailIcon />
      </Badge>

      {/* Dot badge */}
      <Badge variant="dot" color="error">
        <NotificationsIcon />
      </Badge>

      {/* Combined with icon button */}
      <IconButton aria-label="17 new notifications">
        <Badge badgeContent={17} color="error">
          <NotificationsIcon />
        </Badge>
      </IconButton>
    </Stack>
  )
}

// Control badge visibility
function ControlledBadge() {
  const [count, setCount] = React.useState(1)
  const [invisible, setInvisible] = React.useState(false)

  return (
    <Badge badgeContent={count} invisible={invisible} color="primary">
      <MailIcon />
    </Badge>
  )
}
```

## Accessibility

- When combining with IconButton, include the notification count in `aria-label`
- Example: `aria-label="4 new messages"`
- If badge content is only displayed visually, provide alternative text for screen reader users
- For dynamically changing badges, consider using `aria-live` regions
</En>

## 使用例

```tsx
import Badge from '@mui/material/Badge'
import IconButton from '@mui/material/IconButton'
import MailIcon from '@mui/icons-material/Mail'

        <NotificationsIcon />
      </Badge>

      {/* アイコンボタンと組み合わせ */}
      <IconButton aria-label="17件の新しい通知">
        <Badge badgeContent={17} color="error">
          <NotificationsIcon />
        </Badge>
      </IconButton>
    </Stack>

// バッジの表示制御
function ControlledBadge() {
```

## アクセシビリティ

- IconButtonと組み合わせる場合、`aria-label`に通知数を含めてください
- 例：`aria-label="4件の新しいメッセージ"`
- バッジの内容が視覚的にのみ表示される場合、スクリーンリーダーユーザーのための代替テキストを提供してください
- 動的に変化するバッジには、`aria-live`リージョンの使用を検討してください
