# TextField

Text input field

## Overview

The MUI TextField component is a basic form component for users to input text. It supports single-line input, multi-line input (textarea), and various variants (outlined, filled, standard). It comprehensively provides features necessary for forms, including labels, helper text, error messages, and required field marking. Designed with accessibility and usability in mind.

## Usage Example

```tsx
import TextField from '@mui/material/TextField'
import { useState } from 'react'

function ContactForm() {
  const [email, setEmail] = useState('')

  return (
    <TextField
      label="Email Address"
      type="email"
      value={email}
      onChange={(e) => setEmail(e.target.value)}
      required
      helperText="We'll never share your email"
      variant="outlined"
      fullWidth
    />
  )
}
```

## Accessibility

- Provides visual and semantic labels using the `label` property
- Indicates required fields using the `required` property
- Provides error states and messages using `error` and `helperText`
- Specifies appropriate input type using the `type` attribute
- Screen readers properly announce labels, values, and error messages
- Fully supports keyboard operations (Tab, arrow keys, etc.)
