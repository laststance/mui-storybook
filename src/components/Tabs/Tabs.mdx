# Tabs

Tab navigation

## Overview

The MUI Tabs component is a navigation component that switches between and displays multiple tabs of related content. It efficiently organizes content for different views or categories while using screen space effectively. It supports various patterns including horizontal/vertical layouts, scrollable tabs, and tabs with icons. Used in combination with the Tab component.

## Usage Example

```tsx
import Tabs from '@mui/material/Tabs'
import Tab from '@mui/material/Tab'
import Box from '@mui/material/Box'
import { useState } from 'react'

function TabPanel({ children, value, index }) {
  return (
    <div role="tabpanel" hidden={value !== index}>
      {value === index && <Box sx={{ p: 3 }}>{children}</Box>}
    </div>
  )
}

function BasicTabs() {
  const [value, setValue] = useState(0)

  return (
    <>
      <Tabs value={value} onChange={(_, newValue) => setValue(newValue)}>
        <Tab label="Tab One" />
        <Tab label="Tab Two" />
        <Tab label="Tab Three" />
      </Tabs>
      <TabPanel value={value} index={0}>Content 1</TabPanel>
      <TabPanel value={value} index={1}>Content 2</TabPanel>
      <TabPanel value={value} index={2}>Content 3</TabPanel>
    </>
  )
}
```

## Accessibility

- Set `role="tab"` for each tab and `role="tabpanel"` for tab panels
- Supports keyboard operations (arrow keys, Home, End)
- Selected tabs are indicated with `aria-selected="true"`
- Tabs and tab panels are associated using `aria-controls` and `id`
- Screen readers can understand the current tab and available tabs

## 使用例

```tsx
import Tabs from '@mui/material/Tabs'
import Tab from '@mui/material/Tab'
import Box from '@mui/material/Box'
import { useState } from 'react'

function TabPanel({ children, value, index }) {
  return (
    <div role="tabpanel" hidden={value !== index}>
      {value === index && <Box sx={{ p: 3 }}>{children}</Box>}
    </div>
  )
}

function BasicTabs() {
  const [value, setValue] = useState(0)

  return (
    <>
      <Tabs value={value} onChange={(_, newValue) => setValue(newValue)}>
        <Tab label="タブ 1" />
        <Tab label="タブ 2" />
        <Tab label="タブ 3" />
      </Tabs>
      <TabPanel value={value} index={0}>コンテンツ 1</TabPanel>
      <TabPanel value={value} index={1}>コンテンツ 2</TabPanel>
      <TabPanel value={value} index={2}>コンテンツ 3</TabPanel>
    </>
  )
}
```

## アクセシビリティ

- 各タブに`role="tab"`、タブパネルに`role="tabpanel"`を設定
- キーボード操作（矢印キー、Home、End）をサポート
- 選択されたタブは`aria-selected="true"`で示されます
- タブとタブパネルは`aria-controls`と`id`を使用して関連付けられます
- スクリーンリーダーは現在のタブと利用可能なタブを理解できます
