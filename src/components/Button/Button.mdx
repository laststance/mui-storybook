# Button

MUI Button component wrapper. Provides interactive elements for users to perform actions.

## Overview

The Button component wraps Material-UI's Button and provides the following features:

- **Variants**: 3 types - `text`, `outlined`, `contained`
- **Colors**: `primary`, `secondary`, `success`, `error`, `info`, `warning`
- **Sizes**: `small`, `medium`, `large`
- **Accessibility**: WCAG AA compliant color contrast

## Usage Example

```tsx
import Button from '@/components/Button'

// Basic usage
<Button variant="contained" color="primary">
  Click Me
</Button>

// With click handler
<Button
  variant="outlined"
  onClick={() => console.log('clicked')}
>
  Submit
</Button>

// Disabled state
<Button disabled>
  Disabled Button
</Button>
```

## Accessibility

- Always provide clear and descriptive labels for buttons
- Use `aria-label` for icon-only buttons
- Keyboard navigation is supported
</En>

## 使用例

```tsx
import Button from '@/components/Button'

// 基本的な使用法
<Button variant="contained" color="primary">
  クリック
</Button>

// クリックハンドラー付き
<Button
  variant="outlined"
  onClick={() => console.log('clicked')}
```

- ボタンには常に明確で説明的なラベルを付けてください
- アイコンのみのボタンには`aria-label`を使用してください
- キーボードナビゲーションがサポートされています
