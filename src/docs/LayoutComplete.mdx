### The 8px Grid System

MUI's spacing system is based on an **8px grid** by default:

```typescript
// theme.spacing(n) = n * 8px
theme.spacing(1)  // 8px
theme.spacing(2)  // 16px
theme.spacing(3)  // 24px
theme.spacing(4)  // 32px
```

### Quick Decision: Grid vs Flexbox

- **1D layout (row OR column)?** → Use Flexbox (Stack component)
- **2D layout (rows AND columns)?** → Use CSS Grid (Grid component)
- **Content dictates size?** → Flexbox (content-first)
- **Structure dictates size?** → Grid (structure-first)

### Design Quality Checklist

**Before shipping any layout, verify:**

- Proximity — Are related elements grouped together?
- Alignment — Do elements align to a consistent grid?
- Repetition — Are styles consistent throughout?
- Contrast — Is hierarchy clear through size/color differences?
- Responsiveness — Does it work across all breakpoints?
- Accessibility — Is it usable with keyboard/screen reader?
